# Code and Data Documentation for Bachelor Thesis

This repository contains the code documentation for my Bachelor's thesis:

**Rational Voting in Swiss Single-Seat Cantons:  
The Impact of Candidate Numbers and Ideological Positions on Electoral Outcomes**

Author: Ivan Ramon Flühmann  
University of St. Gallen  
Year: 2026

## Purpose of this Repository

This repository provides supplementary documentation for the data construction and empirical analysis conducted in the thesis. It contains the R code used for data cleaning, processing of election data, construction of ideological measures, and statistical analysis.

The purpose of this repository is to improve transparency and reproducibility of the empirical workflow.

## Important Data Availability Note

The final analysis dataset and several raw input datasets cannot be publicly shared.

In particular, the smartvote candidate data used to construct ideological positions were obtained directly from smartvote for the purpose of this thesis. Due to data usage agreements and candidate-level privacy restrictions, these data are not included in this repository.

As a result, the code in this repository is provided for documentation purposes, but it cannot be fully executed without access to the original restricted data sources.
